# NxtTrendz E-Commerce Application

A mini e-commerce web application built using React.

## Features
- Product listing with fixed images
- Product details page
- Add to cart with quantity control
- Cart count in header
- Individual item price & total amount
- Persistent cart using localStorage
- Product search functionality

## Tech Stack
- React
- React Router
- Context API
- CSS

## How to Run
```bash
npm install
npm run dev
